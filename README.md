# Kevin51jiang Tap

## How do I install these formulae?

`brew install kevin51jiang/tap/<formula>`

Or `brew tap kevin51jiang/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kevin51jiang/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
